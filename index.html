Pendahuluan

Penjelasan Umum

Aplikasi ini merupakan suatu aplikasi yang menghubungkan antara user yang sedang melakukan aktivitas liburan dengan user lain, travel agent, maupun pengelola objek wisata.

Memulai dan Mengoperasikan

  1. Download aplikasi temanjalan.co melalui google playstore ke perangkat android anda
  2. Untuk menjalankan aplikasi temanjalan.co tap atau pilih ikon temanjalan.co yang sebelumnya telah didownload melalui google playstore

Registrasi

Untuk menampilkan halaman daftar. Pada tampilan utama tap/pilih menu “registrasi” untuk masuk ke halaman pendaftaran seperti gambar dibawah ini:

Form Registrasi

Untuk melakukan registrasi dilakukan dengan cara mengisi data sebagai berikut:

  • Email : diisi dengan email user, wajib diisi
  • Password : diisi denganpassword, wajib diisi
  • Confirm password : diisi dengan mengisi ulang password yang telah dimasukan sebelumnya
  • Nama lengkap : diisi dengan nama user
  • Kota tempat tinggal : diisi dengan kota tempat tinggal user
  • Nomor handphone : diisi dengan nomor telepon
  • Jenis kelamin : tap/pilih gender User setelah mengisi data-data di atas dengan lengkap.

Login

Jika registrasi berhasil, maka secara otomatis akan beralih ke tampilan utama. Silahkan isi username dan password yang telah anda daftarkan.

Mau Jalan

Fitur ini berguna untuk user yang ingin melakukan perjalanan dan membuka diri untuk user lain bergabung. Tampilan fitur ini adalah sebagai berikut:

User yang ingin melakukan perjalanan harus mendaftarkan rencana perjalanannya dengan mengisi kolom sebagai berikut:

  • Nama trip kamu : diisi dengan nama perjalanan
  • Awal perjalanan kamu : diisi dengan tanggal memulai perjalanan
  • Akhir perjalanan kamu : diisi dengan tanggal selesai perjalanan
  • Tujuan trip kamu : diisi dengan tempat wisata Tujuan
  • Titik kumpul trip kamu : diisi dengan tempat kumpul saat akan berangkat
  • Kuota trip : diisi dengan jumlah maksimal kuota yang dibuka
  • Detail perjalanan : diisi dengan deskripsi singkat tentang perjalanan

<style> p { ... } </style> <p>...</p> /* Class Selector - selects all of the called class */ <style> .class1 { ... } </style> <h1 class="class1">...</h1> /* Id Selector - selects elements with the unique id attribute */ <style> #id1 { ... !important } </style> <div id="id1">...</div> /* Inline Selector - selects individual elements - NOT RECOMMENDED */ <h2 style="color: red;">...</h2>

Riwayat

Untuk melihat riwayat perjalanan yang telah anda ikuti dan riwayat perjalanan yang telah anda buat, maka pada menu join jalan tap/pilih icon teman jalan.

Maka akan muncul menu sebagai berikut:

  • Join trip : daftar riwayat perjalanan yang anda ikuti
  • Tripku : daftar perjalanan yang telah anda buat
